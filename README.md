# roguelike_cli
Un juego Roguelike en consola que implementa hilos.

## Ejecucion

Para correr el programa es necesario ejecutar el siguiente comando.
```
python main.py
```
Los controles para moverse en el juego son los siguientes:
```
w - arriba
a - izquierda
s - abajo
d - derecha
```
